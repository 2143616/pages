🏛 1. 网络模型总览

计算机网络的核心思维是分层——每一层解决特定问题，向上层提供抽象服务。最经典的是OSI 七层模型和TCP/IP 四层模型。

OSI 七层 vs TCP/IP 四层

OSI 七层TCP/IP 四层核心协议数据单元
7 应用层应用层HTTP / HTTPS / FTP / SMTP / DNS / WebSocket消息
6 表示层SSL/TLS（加密）、字符编码转换—
5 会话层RPC、Socket 建立/维护会话—
4 传输层传输层TCP（可靠）、UDP（不可靠）段
3 网络层网络层IP（IPv4/IPv6）、ICMP、ARP、OSPF、BGP包
2 数据链路层网络接口层Ethernet、MAC、VLAN、PPP帧
1 物理层RJ45、光纤、WiFi（802.11）、蓝牙比特

数据封装过程

封装
  ┌───────────────────────────────────────────────────┐
  │ 应用层:  HTTP 请求 "GET /index.html"             │
  ├───────────────────────────────────────────────────┤

  ├───────────────────────────────────────────────────┤

  比特流 → 网线 → 路由器 → 互联网 → 目标服务器

🔌 2. 物理层 & 数据链路层

2.1 物理层

把比特（0/1）转换成信号在介质上传输。关键概念：带宽（bps）、时延、信道复用（频分/时分/波分）。

2.2 数据链路层

负责相邻节点之间的帧传输和差错检测。核心：

📡 MAC 地址

每个网卡有唯一的 48 位物理地址（如 AA:BB:CC:DD:EE:FF）。链路层用 MAC 地址标识设备

🔄 ARP 协议

Address Resolution Protocol：已知 IP → 询问"谁的 IP？" → 对方回复 MAC 地址

🔁 以太网帧

前导码 + 目标MAC + 源MAC + 类型 + 数据(46~1500B) + FCS（CRC 校验）

🔄 VLAN

在帧头加 802.1Q 标签，把物理上的一个交换机隔离成多个"虚拟局域网"

class ARPTable:
    """模拟 ARP 缓存表"""
    def __init__(self):
        self._table = {}  # {IP: MAC}

    def query(self, ip):
        if ip in self._table:
            print(f"ARP 缓存命中: {ip} → {self._table[ip]}")
            return self._table[ip]
        print(f"ARP 请求: 谁的 IP 是 {ip}？")
        # 模拟广播响应
        self._table[ip] = "00:1A:2B:3C:4D:5E"
        print(f"ARP 响应: {ip} 的 MAC 是 {self._table[ip]}")
        return self._table[ip]

arp = ARPTable()

print(arp.query("192.168.1.1"))  # 第二次 → 缓存命中

3.3 路由表与转发

Python
class Router:
    """模拟路由器路由表与转发"""
    def __init__(self, name):
        self.name = name
        self.routing_table = []  # [(网络, 下一跳, 接口)]

    def add_route(self, network, next_hop, interface):
        self.routing_table.append((
            ipaddress.IPv4Network(network, strict=False),
            next_hop, interface
        ))
        # 按前缀长度从长到短排序（最长前缀匹配）
        self.routing_table.sort(key=lambda x: x[0].prefixlen, reverse=True)

    def forward(self, dest_ip):
        ip = ipaddress.IPv4Address(dest_ip)
        for net, next_hop, intf in self.routing_table:
            if ip in net:
                print(f"[{self.name}] 转发 {dest_ip} → 下一跳 {next_hop} (接口 {intf})")
                return
        print(f"[{self.name}] ❌ 无路由: {dest_ip}")

# 构建网络
r1 = Router("R1")
r1.add_route("192.168.1.0/24", "直接", "eth0")
r1.add_route("10.0.0.0/8", "192.168.1.254", "eth0")
r1.add_route("0.0.0.0/0", "192.168.1.1", "eth0")  # 默认路由

r1.forward("192.168.1.50")     # 匹配 /24
r1.forward("10.1.2.3")         # 匹配 /8
r1.forward("8.8.8.8")          # 匹配默认路由

import subprocess
import platform
import time

def ping(host: str, count=4) -> float | None:
    """使用系统 ping 命令测试连通性"""
    param = "-n" if platform.system().lower() == "windows" else "-c"
    cmd = ["ping", param, str(count), host]

    try:
        start = time.time()
        result = subprocess.run(cmd, capture_output=True, text=True, timeout=10)
        elapsed = time.time() - start
        print(f"Ping {host} ({count} 次) 耗时: {elapsed:.2f}s")
        print(result.stdout[-500:])  # 显示尾部
        return elapsed
    except subprocess.TimeoutExpired:
        print("❌ 请求超时")
        return None

def simulate_traceroute(target_ip, hops=8):
    """模拟 Traceroute — 通过 TTL 递增探测路径"""
    print(f"追踪路由到 {target_ip} (最多 {hops} 跳):\n")
    for ttl in range(1, hops + 1):
        # 模拟路由器响应
        if ttl <= 4:
            router_ip = f"10.0.{ttl}.1"
            rtt = round(5 + ttl * 3 + hash(target_ip) % 10, 1)
            print(f"  {ttl:>2}  {router_ip:>15}  {rtt} ms")
        elif ttl == 5:
            router_ip = "203.0.113.45"
            print(f"  {ttl:>2}  {router_ip:>15}  12.3 ms")
        elif ttl == 6:
            print(f"  {ttl:>2}  {target_ip:>15}  21.7 ms  ✅ 到达目标")
            break
        else:
            print(f"  {ttl:>2}  * 请求超时")

📦 4. 传输层（TCP / UDP）

传输层提供端到端通信——给应用层进程之间建立逻辑连接。两个核心协议：

📮 TCP

Transmission Control Protocol
✅ 可靠、有序、面向连接
✅ 流量控制（滑动窗口）
✅ 拥塞控制（慢启动/拥塞避免）
✅ 三次握手 + 四次挥手

📨 UDP

User Datagram Protocol
❌ 不可靠（发出去不管）
❌ 无序、无连接
✅ 低延迟、低开销
✅ 适用：DNS / 视频 / 游戏

4.1 TCP 三次握手

状态机
  客户端                   服务器
    │                        │
    │    SYN=1, seq=x        │
    │───────────────────────→│  LISTEN → SYN_RCVD
    │                        │
    │  SYN=1, ACK=1,         │
    │  seq=y, ack=x+1        │
    │←───────────────────────│  SYN_RCVD → ESTABLISHED
    │                        │
    │    ACK=1, seq=x+1,     │
    │    ack=y+1             │
    │───────────────────────→│
    │                        │
    │    连接建立, 开始传数据  │
    │◄──────────────────────►│  ESTABLISHED

4.2 TCP 拥塞控制模拟

Python
class TCPCongestionControl:
    """模拟 TCP 拥塞控制（慢启动 + 拥塞避免 + 快速重传）"""
    def __init__(self):
        self.ssthresh = 16       # 慢启动阈值
        self.cwnd = 1            # 拥塞窗口（单位：MSS）
        self.rtt = 0              # 轮数
        self.log = []

    def step(self, loss=False):
        self.rtt += 1
        if loss:  # 丢包检测
            print(f"  ❌ RTT {self.rtt}: 丢包! cwnd={self.cwnd} → 1 (重置)")
            self.ssthresh = max(self.cwnd // 2, 2)
            self.cwnd = 1
        else:
            if self.cwnd < self.ssthresh:
                # 慢启动：每 RTT 翻倍
                old = self.cwnd
                self.cwnd *= 2
                print(f"  RTT {self.rtt}: 慢启动 {old} → {self.cwnd} (ssthresh={self.ssthresh})")
            else:
                # 拥塞避免：每 RTT +1
                old = self.cwnd
                self.cwnd += 1
                print(f"  RTT {self.rtt}: 拥塞避免 {old} → {self.cwnd}")
        self.log.append((self.rtt, self.cwnd, self.ssthresh))

# 模拟 12 轮传输，中间遇到一次丢包
tcp = TCPCongestionControl()
for i in range(12):
    tcp.step(loss=(i == 7))

4.3 TCP 状态迁移

Python
class TCPConnection:
    """模拟 TCP 连接状态机"""
    STATES = {
        "CLOSED", "LISTEN", "SYN_SENT", "SYN_RCVD",
        "ESTABLISHED", "FIN_WAIT_1", "FIN_WAIT_2",
        "CLOSE_WAIT", "CLOSING", "LAST_ACK", "TIME_WAIT",
    }
    def __init__(self, role="client"):
        self.state = "CLOSED"
        self.role = role

    def event(self, event):
        transitions = {
            "CLOSED": [("主动打开", "SYN_SENT"), ("被动打开", "LISTEN")],
            "SYN_SENT": [("收到 SYN+ACK", "ESTABLISHED")],
            "LISTEN": [("收到 SYN", "SYN_RCVD")],
            "SYN_RCVD": [("收到 ACK", "ESTABLISHED")],
            "ESTABLISHED": [("主动关闭", "FIN_WAIT_1"), ("收到 FIN", "CLOSE_WAIT")],
            "FIN_WAIT_1": [("收到 ACK", "FIN_WAIT_2"), ("收到 FIN", "CLOSING")],
            "FIN_WAIT_2": [("收到 FIN", "TIME_WAIT")],
            "CLOSE_WAIT": [("关闭", "LAST_ACK")],
            "LAST_ACK": [("收到 ACK", "CLOSED")],
            "CLOSING": [("收到 ACK", "TIME_WAIT")],
            "TIME_WAIT": [("超时", "CLOSED")],
        }
        for e, next_state in transitions.get(self.state, []):
            if e == event:
                old = self.state
                self.state = next_state
                print(f"[{self.role}] {old} ─[{event}]→ {next_state}")
                return
        print(f"[{self.role}] ⚠ 状态 {self.state} 不能处理事件 '{event}'")

# 演示三次握手
c = TCPConnection("Client")
s = TCPConnection("Server")

print("=== 三次握手 ====")
c.event("主动打开")       # CLOSED → SYN_SENT
s.event("被动打开")       # CLOSED → LISTEN
s.event("收到 SYN")       # LISTEN → SYN_RCVD
c.event("收到 SYN+ACK")   # SYN_SENT → ESTABLISHED
s.event("收到 ACK")       # SYN_RCVD → ESTABLISHED

print("\n=== 数据传输中 ===")

print("\n=== 四次挥手 ====")
c.event("主动关闭")       # ESTABLISHED → FIN_WAIT_1
s.event("收到 FIN")       # ESTABLISHED → CLOSE_WAIT
c.event("收到 ACK")       # FIN_WAIT_1 → FIN_WAIT_2
s.event("关闭")           # CLOSE_WAIT → LAST_ACK
c.event("收到 FIN")       # FIN_WAIT_2 → TIME_WAIT
s.event("收到 ACK")       # LAST_ACK → CLOSED
c.event("超时")           # TIME_WAIT → CLOSED

5.3 自己实现简单的 HTTP Server

Python
from http.server import HTTPServer, BaseHTTPRequestHandler
import json

class SimpleHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        print(f"→ GET {self.path}")
        if self.path == "/":
            self.send_response(200)
            self.send_header("Content-Type", "text/plain; charset=utf-8")
            self.end_headers()
            self.wfile.write("你好，计算机网络！\n".encode())
        elif self.path == "/api/info":
            self.send_response(200)
            self.send_header("Content-Type", "application/json")
            self.end_headers()
            data = json.dumps({
                "method": "GET",
                "path": self.path,
                "headers": dict(self.headers),
                "client": self.client_address[0],
            }, ensure_ascii=False)
            self.wfile.write(data.encode())

    def log_message(self, format, *args):
        print(f"  ↪ {args}")

def start_server(port=8888):
    server = HTTPServer(("127.0.0.1", port), SimpleHandler)
    print(f"🌐 HTTP 服务器启动: http://127.0.0.1:{port}")
    print("  按 Ctrl+C 停止")
    try:
        server.serve_forever()
    except KeyboardInterrupt:
        print("\n服务器已停止")
        server.server_close()

if __name__ == "__main__":
    start_server()

import socket

def dns_lookup(domain):
    """DNS 查询：域名 → IP 地址"""
    try:
        ip = socket.gethostbyname(domain)

import threading

def handle_client(conn, addr):
    print(f"[新连接] {addr}")
    with conn:
        while True:
            data = conn.recv(1024)  # 接收最多 1024 字节
            if not data: break        # 客户端关闭
            print(f"  收到: {data.decode()!r}")
            conn.sendall(data)        # 原样发回（echo）
    print(f"[断开] {addr}")

def tcp_echo_server(host="127.0.0.1", port=9999):
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as server:
        server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
        server.bind((host, port))
        server.listen(5)
        print(f"🔊 Echo 服务器: {host}:{port}")
        try:
            while True:
                conn, addr = server.accept()
                threading.Thread(target=handle_client, args=(conn, addr), daemon=True).start()
        except KeyboardInterrupt:
            print("\n服务器关闭")

if __name__ == "__main__":

import socket
import concurrent.futures

def scan_port(host, port):
    """扫描单个端口"""
    try:
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            s.settimeout(1)
            result = s.connect_ex((host, port))
            if result == 0:
                return port
    except:
        pass
    return None

def port_scanner(host, start=1, end=100):
    print(f"🔍 扫描 {host} 的端口 {start}~{end}:")
    open_ports = []
    with concurrent.futures.ThreadPoolExecutor(max_workers=20) as executor:
        futures = {executor.submit(scan_port, host, p): p for p in range(start, end + 1)}
        for future in concurrent.futures.as_completed(futures):
            result = future.result()
            if result:
                service = {
                    22: "SSH", 80: "HTTP", 443: "HTTPS",
                    3306: "MySQL", 6379: "Redis", 8080: "HTTP-Alt",
                }.get(result, "?")
                open_ports.append((result, service))
                print(f"  ✅ 端口 {result:>5} 开放 ({service})")

    if not open_ports:
        print("  ❌ 未发现开放端口")
    return open_ports

import socket
import select
import time

def nonblocking_connect(host, port, timeout=3):
    """非阻塞连接 + select 轮询"""
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.setblocking(False)
    try:
        s.connect((host, port))
    except BlockingIOError:
        pass  # 预期异常 — 连接已在后台进行

    print(f"非阻塞连接 {host}:{port}...")
    _, ready, _ = select.select([], [s], [], timeout)

    if ready:
        print("✅ 连接成功！")
        s.setblocking(True)
        s.close()
        return True
    else:
        print("❌ 连接超时")
        s.close()
        return False

class SimpleCookieSession:
    """模拟 Cookie + Session 机制"""
    def __init__(self):
        self._store = {}    # {session_id: user_data}
        self._cookies = {}  # {domain: {name: value}}

    def login(self, username, password):
        print(f"用户 {username} 登录...")
        import uuid
        session_id = str(uuid.uuid4())
        self._store[session_id] = {"user": username, "login_time": "2026-07-01"}
        # 服务器在响应头 Set-Cookie 下发 session_id
        self._cookies["session_id"] = session_id
        print(f"  → Set-Cookie: session_id={session_id[:8]}...")
        return session_id

    def request(self, path):
        session_id = self._cookies.get("session_id")
        if not session_id or session_id not in self._store:
            return "401 Unauthorized — 请先登录"
        user = self._store[session_id]
        return f"200 OK — 欢迎 {user['user']}，访问 {path}"

        self.tld_servers = {
            "com": {"a.gtld-servers.net"},

            "google.com": {"142.250.80.4"},
        }

        # 检查权威服务器
        if domain in self.auth_servers:

        elif depth == 1:
            print(f"{indent}② 询问 {tld} TLD 服务器...")
        else:
            print(f"{indent}③ 询问权威服务器...")

        return self.resolve(domain, depth + 1)

import random
import time

class UnreliableNetwork:
    """模拟不可靠网络（延迟 + 丢包）"""
    def __init__(self, min_delay=0.05, max_delay=0.3, loss_rate=0.1):
        self.min_delay = min_delay
        self.max_delay = max_delay
        self.loss_rate = loss_rate

    def send(self, packet):
        delay = random.uniform(self.min_delay, self.max_delay)
        time.sleep(delay)

        if random.random() < self.loss_rate:
            return None  # 丢包

        # 模拟比特翻转（极小概率）
        if random.random() < 0.01:
            corrupted = bytearray(packet)
            idx = random.randint(0, len(corrupted) - 1)
            corrupted[idx] ^= 0xFF
            return bytes(corrupted)

        return packet

import itertools

class LoadBalancer:
    """常见负载均衡策略"""
    def __init__(self, servers):
        self.servers = servers
        self._rr = itertools.cycle(servers)
        self._connections = {s: 0 for s in servers}

    def round_robin(self):
        """轮询 — 依次分配"""
        return next(self._rr)

    def least_connections(self):
        """最少连接 — 分配当前连接数最少的"""
        return min(self._connections, key=self._connections.get)

    def weighted(self, weights):
        """加权轮询 — 性能强的服务器分配更多请求"""
        total = sum(weights)
        choices = []
        for server, weight in zip(self.servers, weights):
            choices.extend([server] * weight)
        return random.choice(choices)

12.1 面试必问题

1. 浏览器输入 URL 后发生了什么？

DNS → TCP 三次握手 → TLS 握手(HTTPS) → HTTP 请求 → 服务器处理 → HTTP 响应 → 浏览器渲染

2. TCP 为什么三次握手不是两次？

防止已失效的连接请求到达服务器。两次握手：客户端 SYN 延迟重发 → 服务器误建连接 → 浪费资源

3. HTTP/1.1 vs HTTP/2 vs HTTP/3

1.1: 队头阻塞 / 2: 多路复用 + 二进制分帧 / 3: QUIC (UDP) 解决 TCP 队头阻塞

4. HTTPS 加密过程？

混合加密：非对称(RSA/ECDHE)交换对称密钥 → 对称加密(AES)传输数据

5. Cookie vs Token vs Session

Cookie 客户端存储 / Session 服务端存储 / Token 自包含(JWT)，无状态

6. 从输入 URL 到页面显示完整流程

DNS(1ms) → TCP(1RTT) → TLS(2RTT) → HTTP(1RTT) → 解析HTML+CSS+JS → 渲染

def min_latency_by_distance(distance_km):
    """光在光纤中的速度约 200,000 km/s"""
    speed_in_fiber = 200_000  # km/s
    one_way = distance_km / speed_in_fiber * 1000  # ms
    rtt = one_way * 2
    return one_way, rtt

import requests
from tqdm import tqdm

def download_with_progress(url: str, save_path: str):
    """带进度条的文件下载（展现 HTTP Range 分片）"""
    response = requests.get(url, stream=True)
    total_size = int(response.headers.get("content-length", 0))

    print(f"文件大小: {total_size/1024/1024:.1f} MB")
    print(f"Content-Type: {response.headers.get('content-type')}")
    print(f"Accept-Ranges: {response.headers.get('accept-ranges', '无')}")

    with open(save_path, "wb") as f:
        with tqdm(total=total_size, unit="B", unit_scale=True,
                   desc="下载中", ncols=80) as pbar:
            for chunk in response.iter_content(chunk_size=64*1024):
                if chunk:
                    f.write(chunk)
                    pbar.update(len(chunk))

    print("✅ 下载完成！")
